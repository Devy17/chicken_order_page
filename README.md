# 1. 기획서

## 1.1 프로젝트 개요

### **🍗네네치킨 홈페이지**

[네네치킨](https://nenechicken.com/index.asp)


본 프로젝트는 React와 JSX를 활용하여 네네치킨 홈페이지를 구현하는 것을 목표로 합니다.

그동안 JavaScript(JSX)와 CSS(SCSS)를 배웠고, 특히 React를 활용한 웹 개발을 학습했는데, 

**네네치킨 홈페이지**는 다양한 UI 컴포넌트, 동적 기능, 그리고 복잡한 스타일링이 포함되어 있어

배운 기술들을 실습하고 공부하기에 좋은 사이트라고 생각했기 때문에 본 프로젝트의 주제로 선정했습니다.

구현한 페이지는 **Main 페이지, Login 페이지, MyPage 페이지, 브랜드 창업 페이지** 총 **4페이지** 입니다.

# 2. 요구사항 정의서

## 2.1 필수 구현 기능

- 메인 페이지
    - 배너 슬라이드 구현
    - 메뉴 네비게이션 바 구현
    - 인기 메뉴 소개 섹션 구현
    - 배달/포장을 선택할 수 있는 모달 창 구현
- **로그인 페이지**
    - 회원 로그인 폼 구현
    - 로그인 유효성 검사
    - 자동 로그인 및 아이디 저장 구현
    - 비회원 로그인 폼 구현
    - 약관 조회를 위한 모달 창 구현
- **마이페이지**
    - 사용자 개인정보 표시
    - 회원정보 조회
    - 로그인 관리 기능 구현
- **공통**
    - 공통 Header, Footer 제작
    - 로그인 상태 전역 관리

## 2.2 선택 구현 기능

- **브랜드 창업 페이지**
    - 브랜드 소개 및 정보 표시
    - 창업 관련 정보 제공

## 6.3 최종 결론

우리 팀은 이번 React를 활용한 네네치킨 홈페이지 구현 프로젝트를 통해 계획했던 모든 핵심 기능들을 성공적으로 구현할 수 있었습니다. 특히 배너 자동 슬라이드, 로그인 기능, 인증 시스템 등 JavaScript와 React로 구현한 동적 기능들이 모든 테스트가 정상적으로 완료되어 목표했던 기준을 달성할 수 있었습니다. 

다만 아쉬운 점으로는 페이지의 핵심 기능이라고 할 수 있는 주문 이벤트는 시간과 난이도 문제로 구현을 시도해보지 못했고, 배너 슬라이드 애니메이션이나 가격 표시 디자인 등 몇 가지 개선이 필요한 부분들이 있습니다. 비록 React와 JavaScript를 활용한 복잡한 기능들을 구현하는 과정이 쉽지 않았고 많은 어려움을 겪었지만, 이를 통해 실전 경험을 쌓고 새로운 기술들을 배울 수 있어서 매우 의미 있는 시간이었습니다. 🐔
